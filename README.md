# play-my-mood
This website uses Youtube API to fetch all the songs from yuor Youtube / Youtube Music playlist, and allows you to sort through it for a specific mood or vibe.
You can pick all the hype, sad, or new songs from your playlist, and play them right in the website!

## How to Use
Easy. Just copy the link of your playlist, paste it into the website, describe what type of songs you want to hear, and thats it!

[https://play-my-mood.netlify.app/](https://play-my-mood.netlify.app/)
